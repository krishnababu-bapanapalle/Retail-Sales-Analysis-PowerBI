# Retail Sales Analysis Dashboard using Power BI

## Project Overview
This project analyzes retail sales data using Power BI. The dashboard provides insights into sales performance, profit trends, customer segments, product categories, regional performance, discounts, and shipping efficiency.

## Dataset
The dataset contains retail order details such as order date, ship date, customer details, product category, sales, quantity, discount, and profit.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## Data Cleaning
Performed data cleaning in Power Query:
- Removed duplicate records
- Removed blank rows
- Corrected data types
- Trimmed and cleaned text columns
- Created Delivery Days column
- Created Profit Status column
- Created Discount Category column
- Created Year and Month columns

## Dashboard Pages
1. Executive Summary
2. Product Analysis
3. Customer and Region Analysis
4. Discount and Shipping Analysis

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Total Customers
- Total Quantity
- Average Discount
- Average Delivery Days

## DAX Measures
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders


## Key Insights
- Identified top-performing products and categories.
- Analyzed region-wise and state-wise sales performance.
- Found loss-making products and discount impact on profit.
- Studied customer segment performance.
- Analyzed shipping mode and delivery efficiency.

## Conclusion
This dashboard helps business users understand sales trends, profitability, customer behavior, and operational performance. It supports better decisions in product strategy, discount planning, and regional sales improvement.
